# Audit — Produits psychoactifs

## Décision

Le cours « Produits psychoactifs » est traité comme deuxième cours après le cours Vue validé. La mise en forme n’est pas copiée mécaniquement : elle privilégie les effets, les chaînes de causalité, les tableaux juridiques et les schémas explicatifs.

## Contenu intégré

- 1 route Next.js : `/cours/produits-psychoactifs`.
- 12 images pédagogiques renommées dans `public/cours/produits-psychoactifs/refined/`.
- 20 pages sources complètes conservées dans `public/cours/produits-psychoactifs/source/`.
- 8 tableaux HTML restaurés.
- 13 sections de cours.
- Donnée éditable : `src/features/courses/data/produits-psychoactifs.ts`.

## Choix de mise en forme

- Définitions et familles : cartes texte + schéma.
- Alcool : images de contexte, courbes, tableaux et séquences pédagogiques.
- Sanctions : tableaux HTML responsives.
- Cannabis : image explicative + listes + idées fausses en séquence.
- Médicaments : pictogrammes + tableau de synthèse.
- Sources : bloc repliable `SourcePagesBlock`.

## Point de vigilance

Les chiffres sont repris du support fourni. Le cours garde les pages sources pour vérification et traçabilité.
