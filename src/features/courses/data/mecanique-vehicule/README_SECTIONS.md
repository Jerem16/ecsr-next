# Mécanique du véhicule — découpage des sections

Ce cours est découpé en sections réutilisables pour alimenter ensuite les fiches pédagogiques : motorisation, chaîne cinématique, moteur thermique, circuits annexes, embrayage, boîte de vitesses et pont différentiel.

Les images pédagogiques sont conservées dans `public/cours/mecanique/refined/` et les pages sources dans `public/cours/mecanique/source/`.
